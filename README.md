# ⚔️ 元素对决 - Element Clash

洛克王国风格的卡牌对战小游戏，纯前端实现。

## 运行方式

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:3000` 即可开始游戏。

## 游戏特色

- **8种属性克制**：火、水、草、电、冰、龙、普通、恶（扩展支持岩、飞、毒、钢、超、地、虫、格斗）
- **12只宠物**：每只拥有独特六维属性、4个技能和1个被动特性
- **30个技能**：涵盖攻击、变化、吸血、蓄力等多种类型
- **完整战斗系统**：速度判定、先手值、暴击、异常状态、能力变化
- **AI三级难度**：简单(随机)、中等(克制)、困难(策略)
- **丰富动画**：攻击、受击、伤害飘字、克制特效、胜利粒子
- **音效合成**：Web Audio API 生成的即时音效

## 操作说明

1. 点击 **开始对战** → 选择3只宠物 → 进入回合制对战
2. 每回合选择：出招(4选1) / 切换宠物 / 使用道具
3. 属性克制=2倍伤害，被克制=0.5倍伤害
4. 击败敌方全部3只宠物即获胜！
